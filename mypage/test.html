HTML

超文本标记语言

超级文本标记语言是标准通用标记语言下的一个应用，也是一种规范，一种标准,它通过标记符号来标记要显示的网页中的各个部分。网页文件本身是一种文本文件，通过在文本文件中添加标记符，可以告诉浏览器如何显示其中的内容。

CSS

层叠样式表

HTML实现了网页骨架的搭建，CSS是用层叠样式实现网页的布局和样式设置，好比是化妆。

JavaScript

Web开发脚本语言
